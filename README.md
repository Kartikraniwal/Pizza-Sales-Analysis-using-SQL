# Pizza-Sales-Analysis-using-SQL
This project analyzes pizza sales data using SQL to uncover actionable business insights.  The analysis focuses on understanding sales performance, identifying top-selling and high-revenue pizzas, analyzing order trends over time, and evaluating category-wise revenue contribution to support data-driven business decisions. 

The dataset is organized in a normalized relational database with four tables: orders, order_details, pizzas, and pizza_types.
